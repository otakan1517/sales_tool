# 月次締め業務を効率化する売上集計支援ツール（VBA × Excel）

## 概要
本作品は、売上CSVをExcelに取り込み、Power Queryで整形・マスター結合を行い、  
売上明細、エラー一覧、月次集計、店舗別集計、カテゴリ別集計、部門別集計を確認できるようにしたツールです。

また、VBAを用いて **更新ボタンによる再読込** と **月次集計シートのPDF出力** を実装し、  
月次締め業務で発生しやすい定型作業の負担軽減を想定して作成しました。

---

## 作成目的
Excelを使った月次締め業務では、以下のような手作業が発生しやすいと考えました。

- 売上データの取込
- データ整形
- マスター結合
- 集計表の確認
- 出力資料の作成

そこで本作品では、  
**更新・確認・出力までを一連の流れとして扱える形に整理すること** を目的に作成しました。

- Excel

## 主な機能

### 1. 売上CSVの取込
売上CSVをPower Queryで取り込み、Excel上で扱える形に整えています。

### 2. マスター結合
`product_master.xlsx`、`store_master.xlsx` を用いて必要情報を付与し、確認しやすい売上明細を作成しています。

### 3. エラー一覧
不備のあるデータを確認しやすいように、エラー一覧シートを作成しています。

### 4. 各種集計
以下の集計シートを作成しています。

- 月次集計
- 店舗別集計
- カテゴリ別集計
- 部門別集計

### 5. 更新ボタン
VBAで更新処理マクロを作成し、ボタン操作でデータ更新・再計算ができるようにしています。

### 6. PDF出力
月次集計シートをPDFとして出力できるようにし、定型的な出力作業を簡略化しています。

## 工夫した点
- Power Queryで **取込・整形・結合** を分けて整理したこと
- 集計結果だけでなく、**エラー確認用のシート** も作成したこと
- **更新ボタン** と **PDF出力ボタン** を付け、操作をわかりやすくしたこと
- 正常時だけでなく、**エラー時にもメッセージを表示** するようにしたこと
- マクロ記録で作成したコードを読みながら、少しずつ意味を理解して整理したこと

## フォルダ構成

```text
sales_tool/
├── input/
│   ├── sales_2026_01.csv
│   └── sales_2026_01.xlsx
├── master/
│   ├── product_master.xlsx
│   └── store_master.xlsx
├── output/
│   └── 月次集計.pdf
└── tool/
    ├── 売上集計ツール_マクロ練習前後_backup.xlsm
    ├── 売上集計ツール.xlsm
    └── 売上集計ツール.xlsx

   ``` 
## 各フォルダの役割
- `input/`：売上データの入力ファイルを配置
- `master/`：商品マスター・店舗マスターを配置
- `output/`：PDFなどの出力ファイルを保存
- `tool/`：Excelツール本体や作業用ファイルを配置

## 使い方
1. `input/` に売上データを配置する
2. `tool/売上集計ツール.xlsm` を開く
3. 更新ボタンを押して、Power Queryの再読込と再計算を行う
4. 売上明細、エラー一覧、各種集計シートを確認する
5. 必要に応じてPDF出力ボタンを押し、`output/` に月次集計PDFを出力する

## 補足
- `売上集計ツール.xlsm` を本体ファイルとして使用しています
- `売上集計ツール_マクロ練習前後_backup.xlsm` は作業・確認用のバックアップです
- `売上集計ツール.xlsx` はマクロなしの確認用ファイルとして残しています
